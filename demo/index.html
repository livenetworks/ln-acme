    ln-ajax — AJAX Navigation

    Компонента за AJAX навигација и форм submission. Линкови и форми внатре во data-ln-ajax контејнер автоматски се обработуваат преку fetch.

    <div data-ln-ajax>
    <a href="/users">Корисници</a>
    <form method="POST" action="/users">...</form>
</div>

<!-- JSON одговор -->
{
    "title": "Наслов",
    "content": { "main": "<p>HTML</p>" },
    "message": { "type": "success", "title": "OK", "body": "Зачувано!" }
}
    АтрибутОпис
    data-ln-ajaxКонтејнер — сите <a> и <form> внатре се AJAX
    data-ln-ajax="false"Исклучи конкретен елемент од AJAX обработка

    CSRF token автоматски од <meta name="csrf-token">. History pushState за линкови и GET форми.

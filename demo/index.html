ln-frontend — Компоненти

Документација и живи примери за секоја CSS и JS компонента.

CSS компоненти

  • Typography

    Наслови, параграфи, код, цитати, листи

  • Cards

    .card со header/main/footer, hover ефект, поља

  • Tables

    Dark header, hover accent, section rows, responsive

  • Forms & Buttons

    Input, select, textarea, button варијанти

  • Layout & Grid

    Sidebar, header, grid, stack, container

  • Sections & Breadcrumbs

    .section, .section-card, .breadcrumbs

  • Icons

    SVG data-URI иконки преку CSS pseudo-elements

  • Utilities

    Utility класи за flex, spacing, text, colors

JS компоненти

  • Modal

    data-ln-modal — отвори/затвори, ESC, backdrop

  • Toast

    data-ln-toast — нотификации со иконки

  • Tabs

    data-ln-tabs — hash-aware табови

  • Box & Progress

    data-ln-box collapse + data-ln-progress bar

  • Upload

    data-ln-upload — drag-drop, progress, валидација

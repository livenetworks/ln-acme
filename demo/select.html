Select

TomSelect wrapper — го подобрува стандардниот <select> со search, tagging и create функционалност.

Основи

АтрибутНа елементОпис
data-ln-select<select>Активира TomSelect со default конфигурација
data-ln-select='{"create":true}'<select>JSON конфигурација (мерџира се со defaults)

Default конфигурација

{
    "allowEmptyOption": true,
    "controlInput": null,
    "create": false,
    "highlight": true,
    "closeAfterSelect": true,
    "placeholder": "Select...",
    "loadThrottle": 300
}

Напомена: placeholder атрибутот на <select> се користи ако е поставен.

API

// Рачна иницијализација
window.lnSelect.initialize(selectElement);

// Уништи инстанца (cleanup)
window.lnSelect.destroy(selectElement);

// Земи TomSelect инстанца за директен приступ
var ts = window.lnSelect.getInstance(selectElement);

// TomSelect API
ts.addOption({ value: 'new', text: 'Нова опција' });
ts.setValue('new');
ts.clear();
ts.getItems();

Graceful Degradation

Ако window.TomSelect не постои, компонентата тивко не прави ништо. <select>-от остануваат функционален како обичен браузерски select.

Добра практика: Секогаш вклучи TomSelect пред ln-acme ако го користиш.

Жизнен циклус

При DOM remove

Инстанцата автоматски се уништува (cleanup).

При form reset

Selection се чисти автоматски.

Динамички додадени select елементи

Нови select елементи со data-ln-select се обработуваат автоматски.

HTML структура

<!-- Обичен TomSelect -->
<select data-ln-select>
    <option value="">Избери...</option>
    <option value="option1">Опција 1</option>
    <option value="option2">Опција 2</option>
</select>

<!-- TomSelect со create опција -->
<select data-ln-select='{"create":true}' 
        placeholder="Напиши или избери...">
    <option value="opt1">Опција 1</option>
    <option value="opt2">Опција 2</option>
</select>

<!-- TomSelect во форма -->
<form>
    <label for="my-select">Избор</label>
    <select id="my-select" data-ln-select name="role">
        <option value="user">User</option>
        <option value="admin">Admin</option>
    </select>
</form>

Зависности

Peer dependency: TomSelect мора да е вчитана пред ln-acme.

<!-- TomSelect CSS -->
<link href="..." rel="stylesheet">

<!-- TomSelect JS (пред ln-acme!) -->
<script src="..."></script>

<!-- ln-acme (со вградена поддршка за TomSelect) -->
<link rel="stylesheet" href="dist/ln-acme.css">
<script src="dist/ln-acme.iife.js"></script>

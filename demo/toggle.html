Toggle

Генеричка toggle компонента — додава/тргни .open класа на елемент. Работи за sidebar, collapsible, dropdown — било што.

Основи

АтрибутНа елементОпис
data-ln-toggleцелен елементКреира инстанца, почнува затворен
data-ln-toggle="open"целен елементПочнува отворен (добива класа .open)
data-ln-toggle-for="id"копчеРеферира на целниот елемент по ID
data-ln-toggle-action="open|close"копчеЕксплицитна акција (default: toggle)

Instance API

var el = document.getElementById('my-element');

el.lnToggle.open();      // Отвори
el.lnToggle.close();     // Затвори
el.lnToggle.toggle();    // Toggle
el.lnToggle.isOpen;      // boolean (read-only)

Events

EventBubblesDetail
ln-toggle:openда{ target: HTMLElement }
ln-toggle:closeда{ target: HTMLElement }
 
document.addEventListener('ln-toggle:open', function (e) {
    console.log('Отворен:', e.detail.target.id);
});

document.addEventListener('ln-toggle:close', function (e) {
    console.log('Затворен:', e.detail.target.id);
});

HTML структура

<!-- Елемент кој ќе се мени -->
<aside id="my-sidebar" data-ln-toggle>
    Содржина
</aside>

<!-- Копче кое го контролива -->
<button data-ln-toggle-for="my-sidebar">
    Тоглирај
</button>

<!-- Почнува отворен -->
<section id="panel" data-ln-toggle="open">
    Отворена содржина
</section>

<!-- Експлицитна акција -->
<button data-ln-toggle-for="panel" 
        data-ln-toggle-action="close">
    Затвори
</button>

CSS за анимации

ln-toggle додава/отстранува класа .open. За CSS анимации користи го collapsible pattern-от:

// Sidebar — transform анимација
.sidebar {
    transform: translateX(-100%);
    @include transition;
    &.open { transform: translateX(0); }
}

// Collapsible — grid-template-rows (ПРЕПОРАЧАНО)
// Користи .collapsible класа или @include collapsible
<section id="panel" data-ln-toggle class="collapsible">
    <article class="collapsible-body">
        Содржина...
    </article>
</section>

// SCSS:
.collapsible      { @include collapsible; }
.collapsible > *  { @include collapsible-content; }

// Или семантички во проект SCSS:
#panel         { @include collapsible; }
#panel > .body { @include collapsible-content; }

Динамички DOM

Компонентата користи MutationObserver — автоматски ги иницијализира сите нови елементи додадени во DOM, вклучувајќи динамички HTML од AJAX или JS.

// Нема потреба од рачна иницијализација.
// MutationObserver автоматски ги открива новите data-ln-toggle елементи.

fetch('/partial').then(r => r.text()).then(html => {
    document.getElementById('content').innerHTML = html;
    // lnToggle автоматски ги иницијализира новите елементи
});

window.lnToggle е достапен само за нестандардни случаи (Shadow DOM, iframe) каде MutationObserver не набљудува.

Progress

Progress bar компонента — визуелен индикатор за напредок. Реактивен: автоматски се ажурира кога атрибутите се менуваат.

Основи

АтрибутНа елементОпис
data-ln-progresstrack (празно)Го означува progress track контејнерот
data-ln-progress="50"barТековна вредност на progress bar
data-ln-progress-max="100"barМаксимална вредност (default: 100)

CSS класи за стилизирање

КласаБојаПримена
.greenSuccess (зелена)<div data-ln-progress="80" class="green">
.yellowWarning (жолта)<div data-ln-progress="50" class="yellow">
.redError (црвена)<div data-ln-progress="20" class="red">
(без класа)Default primary<div data-ln-progress="60">

Пример — Единечни progress bar

  • Успех (80%)

  • Внимание (50%)

  • Грешка (25%)

  • Default (60%)

Пример — Со максимална вредност

  • Storage: 7.5 GB од 10 GB

  • Backup: 45 од 50 items

Пример — Stacked progress (повеќе сегменти)

  • Ресурси (%) — CPU, RAM, Disk

  • Задачи — 4 завршени, 2 во тек, 1 чека

Реактивност — напредок се ажурира

  • Клик за промена на вредност:

HTML структура

<!-- Едноставен progress bar -->
<div data-ln-progress>
    <div data-ln-progress="75" class="green"></div>
</div>

<!-- Со максимална вредност -->
<div data-ln-progress>
    <div data-ln-progress="7" data-ln-progress-max="10"></div>
</div>

<!-- Stacked (повеќе барови) -->
<div data-ln-progress>
    <div data-ln-progress="40" class="green"></div>
    <div data-ln-progress="35" class="yellow"></div>
    <div data-ln-progress="25" class="red"></div>
</div>

Пресметка на ширина

Formula: width = (value / max) * 100%

ValueMaxWidth
7510075%
71070%

CSS детали

Заоблени агли

Последниот bar во stacked progress добива заоблен десен агол.

Реактивност

Компонентата користи MutationObserver за следење на data-ln-progress и data-ln-progress-max атрибутите.

API

// Рачна иницијализација
window.lnProgress(document.body);

// За промена на напредок динамички:
var bar = document.getElementById('my-bar');
bar.setAttribute('data-ln-progress', '50');
bar.setAttribute('data-ln-progress-max', '100');

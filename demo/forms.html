Forms & Buttons

CSS Grid + wrapping <label> pattern. Нема <div class="form-group"> или <div class="form-row">. Grid-от го контролира layout-от преку SCSS — не преку HTML.

Принцип

Секое поле е <label> кој ги обвиткува текстот и input-от (implicit association — без for/id). Root елементот (<form> или <section>) добива @include form-grid во SCSS — 6-колонска решетка, колапсира на 1 колона на мобилен. Column spans се поставуваат на > label:nth-child(N) во app-specific SCSS — никогаш inline style="".

Погрешно — стар pattern

<!-- div wrapper-и, explicit for/id -->
<div class="form-group">
  <label for="name">Ime</label>
  <input type="text" id="name">
</div>

<div class="form-row">
  <div class="form-group">
    <label for="city">Grad</label>
    <input type="text" id="city">
  </div>
  <div class="form-group">
    <label for="zip">Zip</label>
    <input type="text" id="zip">
  </div>
</div>

Точно — нов pattern

<!-- label wrap, grid во SCSS -->
<form id="my-form">

  <label>
    Ime
    <input type="text" name="name">
  </label>

  <label>
    Grad
    <input type="text" name="city">
  </label>

  <label>
    Zip
    <input type="text" name="zip">
  </label>

</form>
Елемент Правило
Root <form id="..."> — styled со @include form-grid во SCSS (6 cols → 1 col на мобилен)
Деца Директно <label> — без <div class="form-group"> или <div class="form-row">
Label Го обвиткува текстот + input (implicit association — не треба for/id). Checkbox/radio: текстот оди после input-от
Column spans Во SCSS преку > label:nth-child(N) { grid-column: span N; } — НИКОГАШ inline style="". span 3 = ½, span 2 = ⅓, span 6 = full
Грешки <small class="text-error"> внатре во <label> + small { display: block } во SCSS
Checkbox / Radio input[type="checkbox"] { width: auto } — ln-acme сетира input { width: 100% } глобално
<fieldset> САМО за семантички поврзани полиња (пр. адреса: улица + град + поштенски). НЕ за визуелни редици
.form-actions Компонентна класа — останува во HTML. Добива grid-column: span 6 (full-width) во SCSS

Имплементација

HTML

<form id="my-form">

  <!-- Стандардно поле — span 3 (половина) преку SCSS -->
  <label>
    Ime <span class="text-error">*</span>
    <input type="text" name="fname" required placeholder="...">
    <small class="text-error">Задолжително поле</small>
  </label>

  <!-- Select — ист pattern -->
  <label>
    Kategorija
    <select name="category">
      <option value="a">Опција А</option>
    </select>
  </label>

  <!-- Checkbox — текстот ПОСЛЕ input-от -->
  <label>
    <input type="checkbox" name="confirmed" value="1">
    Ги потврдувам податоците
  </label>

  <!-- form-actions е компонентна класа, останува во HTML -->
  <div class="form-actions">
    <button type="button" class="btn btn--secondary">Откажи</button>
    <button type="submit" class="btn">Зачувај</button>
  </div>

</form>

SCSS

#my-form {
  @include form-grid;       // 6 cols, gap, responsive (<768px → 1 col)

  > label {
    grid-column: span 3;    // default: половина ширина
  }

  // Специфични spans
  > label:nth-child(3) { grid-column: span 4; }   // 2/3 ширина
  > label:nth-child(4) { grid-column: span 2; }   // 1/3 ширина

  // Три во ред (span 2 × 3 = 6 cols)
  > label:nth-child(5),
  > label:nth-child(6),
  > label:nth-child(7) { grid-column: span 2; }

  // Full-width
  > label:nth-child(8),
  > label:nth-child(10),
  > .form-actions { grid-column: span 6; }

  // Checkbox/radio fix
  input[type="checkbox"],
  input[type="radio"] { width: auto; }

  // Error display
  small { display: block; }
}

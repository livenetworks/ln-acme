Accordion

Wrapper компонента која ги координира toggle елементи. Само еден панел може да биде отворен во исто време.

Основи

АтрибутНа елементОпис
data-ln-accordionparentКреира accordion wrapper
data-ln-toggleпанелДетски toggle елемент
data-ln-toggle-forкопчеРеференцира до панелот по ID

Пример — Основен Accordion

  • Што е ln-acme?

    ln-acme е унифициран frontend library за LiveNetworks проекти. Содржи SCSS CSS framework + vanilla JS компоненти, без зависности.

  • Како да се користи?

    Можеш да ја користиш како npm пакет, преку git submodule, или директно со <link> и <script> тагови.

    import 'ln-acme';
import 'ln-acme/dist/ln-acme.css';
  • Има ли зависности?

    ln-acme е полностојно независен. Сите JS компоненти се напишани во vanilla JavaScript користејќи IIFE pattern.

    Исклучок: ln-select зависи од TomSelect (peer dependency).

API & Events

Constructor

// Рачна иницијализација
window.lnAccordion(document.body);

Events

Accordion слуша на ln-toggle:open events од decendent toggle елементи и затвора ги останатите панели.

EventКога
ln-toggle:openКога еден панел се отвора
ln-toggle:closeКога еден панел се затвора

HTML структура

<ul data-ln-accordion>
    <li>
        <header data-ln-toggle-for="panel1">
            <strong>Наслов 1</strong>
        </header>
        <section id="panel1" data-ln-toggle="open" class="collapsible">
            <section class="collapsible-body">
                <p>Содржина 1 — почнува отворена</p>
            </section>
        </section>
    </li>

    <li>
        <header data-ln-toggle-for="panel2">
            <strong>Наслов 2</strong>
        </header>
        <section id="panel2" data-ln-toggle class="collapsible">
            <section class="collapsible-body">
                <p>Содржина 2 — почнува затворена</p>
            </section>
        </section>
    </li>
</ul>

Зависности

Accordion зависи од ln-toggle компонента. ln-toggle мора да е вчитана пред accordion.
